# Aura Capital Asset | dcondo Ping Property Listing

A premium, high-performance static website for showcasing and selling luxury condominium units in Chiang Mai, Thailand. Specifically featuring dcondo Ping by Sansiri.

## 🌟 Features

- **Direct Owner Listing**: Personalized reach-out for potential buyers without agency overhead.
- **Multilingual Support**: Fully localized in English, Thai, Chinese, Lao, Japanese, and Korean.
- **High Performance**: Vanilla HTML/CSS/JS for near-instant load times and smooth animations.
- **SEO Optimized**: Complete metadata, Open Graph tags, robots.txt, and sitemap.xml.
- **Responsive Design**: Tailored experience for mobile, tablet, and desktop devices.
- **Dynamic Gallery**: Categorized property gallery with integrated lightbox.

## 🚀 Getting Started

### Prerequisites

You only need a local server to run the project. `live-server` is recommended for development.

### Installation

1. Clone the repository.
2. Install dependencies (optional, for development tools):
   ```bash
   npm install
   ```

### Local Development

Run the following command to start a local development server with live reloading:

```bash
make dev
```
Alternatively, use `npm run dev`.

## 📦 Deployment

This website is designed to be hosted on **GitHub Pages**.

To deploy changes:
1. Ensure your changes are committed.
2. Run the deployment command:
   ```bash
   make deploy
   ```
This will push the changes to the `main` branch, triggering the GitHub Actions workflow.

## 🛠 Tech Stack

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Typography**: [Outfit](https://fonts.google.com/specimen/Outfit) via Google Fonts.
- **Icons**: [Font Awesome 6](https://fontawesome.com/).
- **Tools**: GNU Make, GitHub Actions.
